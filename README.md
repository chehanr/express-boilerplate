# express-boilerplate

Minimal express.js server to get things started.

Includes:

- Middleware and `Error` class based error responses.
- Zod parser middleware.
- Request/ response context.

## Running

```ts
npm i
npm run build
npm run start
```

## Development

```ts
npm run dev
```
